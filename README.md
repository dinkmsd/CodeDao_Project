# English App

Helper Learning App project.
This project is a starting point for a Flutter application.

## Tasks and Features
* Improve the UI
* Implementing the App with Features
* Login Page using Authentication
* Work on the improvement and building the various pages of the app
* The backend is coded using Node.js and MongoDB.
* Database integration with MongoDB

## Setup

  ##### Clone the repository
```bash
git clone https://github.com/dinkmsd/CodeDao_Project.git
```
  ##### Move to the desired folder
```bash
cd \CodeDao_Project
```

  ##### To run the app, simply write
```bash
flutter run
```
